# Wedding Landing (Vite + Vue 3 + Tailwind)

Quick start:

```bash
npm install
npm run dev
```

Notes:
- Tailwind is configured in `tailwind.config.cjs` and `src/styles/index.css`.
- Main entry: `src/main.js`, root component: `src/App.vue`.
- Components are separated under `src/components/` by section: `Header`, `Countdown`, `Info`, `Gallery`, `RSVPForm`, `Footer`.

Next steps:

- Hook RSVP form to an API or Google Forms.
